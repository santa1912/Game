# Blackjack Game (Java)

A simple Blackjack card game written in Java using Object-Oriented Programming (OOP).

## Features

* Player vs Computer
* Random card generation
* Automatic computer draw when score is below 17
* Winner checking system
* Interactive console input using Scanner

## Files

* `Blackjack.java` → game logic
* `Main.java` → program entry point

## How to Run

Compile the files:

```bash
javac Main.java Blackjack.java
```

Run the program:

```bash
java Main
```

## Example Output

```text
You : 5 8
Computer : 10 4
Want another card? (y/n): y
You : 5 8 3
```
